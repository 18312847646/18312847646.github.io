搭建Python开发环境01——在Windows下安装Python

发表于 2015-10-15 更新于 2020-04-29 分类于 Python

　　大家好，今天我要带给大家的是在 Windows下安装 Python。

安装 Python

　　官网下载 Python 最新版。

　　开始安装，勾选Add Python 3.5 to PATH。选择自定义安装。

　　默认一直next。到这里看个人，我是全部都用得上所以全选了。

　　安装完试试 DOS 命令：python。变成>>>说明成功了。
