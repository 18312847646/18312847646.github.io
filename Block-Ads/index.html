屏蔽广告

发表于 2016-08-01 更新于 2020-04-29 分类于 技术

　　不定期更新软件去广告的方法。

QQ 去广告

　　最近 SVIP 过期了，QQ突然出来一大堆广告，简单 Google 一下就找到了解决方法。

　　Win+R => %AppData%\Tencent\QQ\Misc => 删除com.tencent.advertisement文件夹下的所有东西（文件夹本身不要删除） => com.tencent.advertisement属性 => 安全 => 高级 => 禁用继承 => 从此对象中删除所有已继承的权限 => 确定 => 是 => 确定。

　　QQ 这下干净了，不过左下角的广告依然无法清除，希望知道的朋友留言告诉我方法。（好吧，我知道今天的博客是滥竽充数的_(:зゝ∠)_）

暴风影音去广告

2017.06.28更

　　Win+R => C:\ProgramData\Baofeng\StormPlayer\Profiles => 删除vod和md文件夹 => 新建vod和md文件。
