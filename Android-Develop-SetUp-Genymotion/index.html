搭建Android开发环境05——Genymotion

发表于 2015-11-02 更新于 2020-04-29 分类于 Android开发

　　Genymotion 是一套完整的工具，它提供了 Android 虚拟环境。它简直就是开发者、测试人员、推销者甚至是游戏玩家的福音。

安装 VirtualBox

　　由于 Genymotion 是基于 VirtualBox 的，所以我们先安装 VirtualBox。进入官网点击大大的download。

　　点击x86/amd64。

　　安装 VirtualBox 基本上一路默认，然后安装虚拟网卡，这里点击安装。

安装 Genymotion

　　安装完 VirtualBox 就开始安装 Genymotion 了。进入官网。找到Get started with Genymotion，点击download。

　　选不带 VirtualBox 的版本下载。

　　有账号的登陆，没账号的注册。

　　安装完打开，打开设置。Account 登陆刚才注册的账号。

　　ADB 选择自己的 SDK 目录。

在 Android Studio 设置 Genymotion

　　在设置里打开 Plugins。

　　搜索 Genymotion，安装完重启 Android Studio。

　　打开设置，在 Genymotion 里面设置路径即可。

在 Android Studio 使用 Genymotion

　　在工具栏点击手机图标。

　　点击 new 即可新建虚拟机。
